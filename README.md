# Classmate - Pro Edition 🎓🎮

A gamified, interactive educational web application designed for students and schools to track progress and engage in competitive learning modules. This project was developed from scratch as a product concept to align technical frontend architecture with core business and user validation logic.

## 🚀 Key Features & Product Logic
* **Rivals Matchmaking System:** Simulates a real-time competitive queue using asynchronous JavaScript (`setInterval`/`setTimeout`).
* **Dynamic Component Rendering:** Multi-page profile slider utilizing dynamic DOM manipulation and state tracking.
* **Client-Side Data Validation:** Robust logical validation structures within application forms to capture and handle improper user inputs.
* **Responsive Web Design (RWD):** Mobile-first UI scaling achieved via fluid CSS Media Queries.

---

## 🛠️ Tech Stack & Architecture
* **Frontend Core:** Semantic HTML5, Vanilla JavaScript (ES6+), Advanced CSS3 (Custom Variables, Keyframe Animations, Flexbox).
* **Environment & Workflows:** Visual Studio Code, Git, GitHub Actions/Pages.

---

## 🧪 QA & Testing Practices Applied
As a Quality Assurance specialist, this application was heavily tested during development using **Chrome DevTools**:
1.  **Functional Testing:** Validated edge cases for timer execution, state changes, and array bounds for the dynamic tips system.
2.  **UI/UX & Compatibility Testing:** Inspected DOM elements and layouts across simulated mobile viewports (Responsive Design validation).
3.  **Log & Console Analysis:** Monitored execution states and cleared potential memory leaks from repeating intervals.
